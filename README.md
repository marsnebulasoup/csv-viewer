# About

The (world's üòé) Simplest (and probably most useless) CSV Viewer.

Load CSV files, or paste in raw data, and view it in a nicely formatted, sortable, searchable, table. 

(Over)made with [Vue](https://vuejs.org), [Vuex](https://vuex.vuejs.org/), [Vuetify](https://vuetifyjs.com/), [Electron](https://www.electronjs.org/), [Vue CLI Plugin Electron Builder](https://nklayman.github.io/vue-cli-plugin-electron-builder/), [Papaparse](https://www.papaparse.com/), [Lodash](https://lodash.com), and üíõ.

Created by Kevin Bryniak ([@marsnebulasoup](https://github.com/marsnebulasoup))

---


## Build / Modify this project

### Project setup
```
npm install
```
### Compiles and hot-reloads for development 
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Customize configuration
See [Configuration Reference](https://cli.vuejs.org/config/).
